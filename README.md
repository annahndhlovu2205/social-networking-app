# Social Networking Web Application

## Overview
A full-stack web application that allows users to register, authenticate, and interact dynamically.

## Features
- User registration & authentication
- RESTful API integration
- Dynamic content updates using AJAX
- JSON-based client-server communication

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB

## Key Learning Outcomes
- Backend development using Node.js & Express
- Database integration with MongoDB
- Client-server architecture
- API design and routing
